JavaScript语言核心

初始化表达式 上面代码中通过[]定义数组元素和通过{}定义对象属性名和属性值之间的映射关系的语法称为初始化表达式。 表达式是JavaScript中的一个短语，这个短语可以通过运算得出一个值。

函数 带有名称和参数的JavaScripte代码段，可以一次定义多次调用。 当将函数和对象合写在一起时，函数就变成了方法（method）。

示例：

		
 	/*1、变量*/
 	var x;   //声明变量x
 	x = 0;   //变量赋值
 	window.console.log("变量赋值结果"+x);   //通过变量获取值
 	
 	/*2、数据类型*/
 	x = 1;

 	book.contents = {};       //{}是一个空对像，它没有属性

 	/*5、函数*/
 	function plus1(x){
 		return x+x;
 	}

 	var square = function(x){
 		return x*x;
 	}
 	square(plus1(5));            // =>100  在一个表达式中调用两个函数
 	
 	
 	/*6、方法*/
 	var a = [];

 	a.reverse();    //将数组元素次序反转方法
