伪类： 专门用来表示元素的一种特殊的状态，比如：访问过的超链接，普通的超链接，获取焦点的文本框；（相当于android中的selector） 当我们需要为处再这些特殊状态的元素设置样式时，就可以使用伪类。

  • : link：表示普通的链接（没用访问过的链接）
  • : visited：表示访问过的链接，浏览器是通过历史记录来判断一个链接是否访问过， 由于涉及到用户的隐私问题，所以使用visited伪类只能设置字体的颜色
  • : hover：表示鼠标移入的状态
  • : active：表示超链接被点击的状态

    • :hover和:active不仅仅对超链接a标签设置，还可以为其他元素设置，IE6中，不支持对超链接以外的元素设置:hover和:active

  • : focus：获取焦点

  • : before：指定元素前
  • : after：指定元素后

  • ::selection：选中的元素

  • : first-child ：第一个子对象
  • : last-child：最后一个子元素
  • : nth-child：可以选中任意位置的子元素，该选择器后边可以指定一个参数“(位置)”，指定需要选中第几个子元素， even表示偶数位的子元数，odd表示奇数位的子元数

  • : first-child ：第一个子对象
  • : last-child：最后一个子元素
  • : nth-child：可以选中任意位置的子元素，该选择器后边可以指定一个参数“(位置)”，指定需要选中第几个子元素， even表示偶数位的子元数，odd表示奇数位的子元数

  • : first-of-type ：同一类型中排第一位的元素
  • : last-of-type：同一类型中排最后一个的元素
  • : nth-of-type：同一类型中排指定位置的元素

伪类选择器：用来选中某个标签的某个状态

标签名 : 伪类{}

示例：

<!--html--> <a href="http://www.baidu.com">访问过的链接</a><br /> <a href="http://www.baidu123456.com">没有访问过的链接</a><br /> <p>我是一个段落</p> <input type="text" /> <!--css--> a:link{ <!--为没有点击过的链接设置字体颜色--> color: yellowgreen; background-color: yellow; font-size:25px; } a:visited{ <!--为访问过的链接设置红色，visited只能设置字体颜色--> color: red; background-color: blue; font-size:15px; } <!--：hover和：active也可以为其他元素设置，IE6中不支持对超链接以外的元素设置这两个伪类--> p:hover{ <!--鼠标移入--> background-color: yellow; } p:active{ <!--被点击的状态--> color: blue; } <!--文本框获取焦点以后，修改背景颜色为黄色--> input:focus{ background-color: yellow; } <!--为p标签中选中的内容使用样式，可以使用::selection伪类； 注意：这个伪类在火狐中需要采用另一种方式编写::-moz-selection, 为了兼容，可以写两份 --> <!--兼容大部分浏览器--> p::selection{ background-color: orange; } <!--兼容火狐--> p::-moz-selection{ background-color: orange; } <!--选中某个子元素--> p:first-child{ <!--选中所有p标签中，是第一个子元素的p标签--> background-color: orange; } p:last-child{ <!--选中所有p标签中，是最后一个子元素的p标签--> background-color: orange; } p:nth-child(3){ <!--选中所有p标签中，是第三个子元素的p标签--> background-color: orange; } p:nth-child(odd){ <!--选中所有p标签中，是奇数位子元素的p标签--> background-color: orange; }

结果展示：

访问过的链接
没有访问过的链接

我是一个段落


否定伪类选择器：可以从已经选中的元素中剔除某些元素

标签名 : 伪类{}

示例：

<!--html--> <span>我是一个段落</span> <span class="hello">我是一个段落</span> <input type="text" /> <span>我是一个段落</span> <span>我是一个段落</span> <!--css--> span:not(.hello){ <!--选中除了class值为hello的所有p元素--> background-color: orange; }

结果展示：

我是一个段落 我是一个段落 我是一个段落 我是一个段落
