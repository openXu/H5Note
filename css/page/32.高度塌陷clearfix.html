子元素和父元素相邻的垂直外边距会发生重叠，子元素的外边距会传递给父元素， 使用空的table标签可以隔离父子元素的外边距，阻止外边距的重叠。

示例：

<!--css--> /* * 经过修改后的clearfix是一个多功能的， * 既可以解决高度塌陷，又可以确保父元素和子元素的垂直外边距不会重叠 */ .clearfix:before, .clearfix:after{ content: ""; display: table; clear: both; } <!--IE6不支持after伪类，需要使用hasLayout做兼容--> .clearfix{ zoom: 1; } <!--html--> <div class="box3"> <div class="box4"> </div> <div class="box1"> <div class="box2"> </div>
