内联元素不能设置width和height；

内联元素可以设置水平方向的内边距；

内联元素可以设置垂直方向内边距，但是不会影响页面的布局；

内联元素可以设置边框，但是垂直的边框不会影响到页面的布局；

内联元素支持水平方向的外边距；

内联元素支持水平方向的外边距，水平方向外边距不会重叠，而是求和；

内联元素不支持垂直方向的外边距；

示例：

<!--css--> span{ background-color: yellow; } .s1{ /*内容区、内边距、边框、外边距*/ /*内联元素不能设置width和height*/ width: 200px; height: 200px; /*内联元素可以设置水平方向的内边距*/ padding-left: 50px; padding-right: 50px; /*内联元素可以设置垂直方向内边距，但是不会影响页面的布局*/ padding-top: 20px; padding-bottom: 50px; } /*内联元素可以设置边框，但是垂直的边框不会影响到页面的布局*/ .s2{ border: 10px blue solid; } .s3{ /*内联元素支持水平方向的外边距*/ margin-left: 50px; margin-right: 50px; /*内联元素不支持垂直方向的外边距*/ margin-top: 50px; margin-bottom: 50px; } .div1{ width: 100px; height: 100px; background-color: red; } <!--html--> <span class="s1">我是span</span> <span class="s2">我是span</span> <span class="s3">我是span</span> <span>我是span</span> <div class="div1"></div>
