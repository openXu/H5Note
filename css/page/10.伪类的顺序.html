伪类的顺序

伪类用于选择某个标签的某种状态，由于伪类不止一个，所以伪类选择器之间可能存在重复选中的时候

比如

涉及a的伪类一共由四个（定义顺序必须严格按照下面顺序）：

  • : link
  • : visited
  • : hover
  • : active

    • 这四个选择器的优先级是一样的。一个链接要么被点击过，要么没被点击，鼠标点击的时候也是移入状态，如果选择器定义位置不注意，可能导致样式不起效

示例：

<!--html--> <a href="http://www.baidu.com"></a> <a href="http://www.baidu122.com"></a> <!--css--> a:link{ color: black; } a:visited{ color: yellowgreen; } a:hover{ background-color: yellow; } a:active{ background-color: red; }
